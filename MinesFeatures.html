Beauty of the Game


Simply click anywhere on the board and watch the beautiful tile animation flood through the opened tiles. Opened tiles display the number of bombs touching them in a subtle white font over uniquely colored tiles. Each unopened tile looks 3D to give opened tiles a pressed in effect.

Same Old Feel


Click the menu Choose from the 3 default difficulties or choose custom to make the board which ever dementions you want with any amount of bombs. Right click to flag a tile or turn on question marks in the game menu to mark tiles unknown by right clicking a flagged tile. By left clicking a tile with correctly marked and amount of flags, all the tiles touching it which are not flagged will be opened. Giving this game a huge edge over other implementations of the game.

Easy Correct Check


Want to make sure you're on the right track? Just press the 'C' key and a green or red overlay covers the entire board. Green indicating all the tiles that have been flagged are bombs and red indicating that at least one tile is incorrectly flagged.

A Classic Look


Select compact in the game menu and quickly be overwhelmed with nostalgia. This mode gives the game a classic feel of how it was first implemented as. Compact mode also looks better on screens with lower resolutions.

Helpful Hints


Whenever stuck in the game either simply click hint in the help menu or press the 'H' key down. If there is an available move (without guessing) A red or yellow box will surround a certain tile on the board. A red box indicates that all tiles touching it should be flagged and a yellow box indicates that the flags surrounding the tile match the bombs surrounding number.

Agonizing Defeats


If by chance the user hits one of the bombs on the board, they all immediately start firing off. One by one, each unmarked bomb appears on the screen. The board fills up with all the bombs very fast. By clicking anywhere on the board or by pressing 'R' (to reset), the game will start over on that same difficulty.

Rewarding Victory


Nothing feels better than clicking the last tile, seeing the remaining unmarked flags get flagged, time stop, and see the victory message. But it's not over, now for the next challenge! Up the intensity by selecting the next difficulty or challenge yourself with custom bigger boards with more bombs. More of a speed guy? Keep trying to improve your time.
